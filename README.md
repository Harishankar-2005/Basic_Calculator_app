# Calculator

A clean, responsive calculator built with HTML, CSS, and vanilla JavaScript.

## Features

- Basic operations: addition, subtraction, multiplication, division
- Percentage (%) and sign toggle (+/−)
- Decimal support
- Keyboard support (`0–9`, `+`, `-`, `*`, `/`, `.`, `Enter`, `Backspace`, `Escape`)
- Error handling (invalid expressions, division by zero)
- Responsive design — works on mobile and desktop

## How to Use

Just open `calculator.html` in any browser. No setup or dependencies required.

## Project Structure

```
calculator.html   # Markup + JavaScript logic
style.css         # Styling and layout
README.md         # This file
```

## Tech Stack

- HTML5
- CSS3 (CSS Grid, CSS Variables, Google Fonts)
- Vanilla JavaScript (no libraries)
